Funciones y Características

Credinet Web

Credinet Web es el sistema electrónico de interacción entre su Empresa y el Banco de Crédito BCP, que le permite realizar sus transacciones financieras y consultar información detallada de las mismas con la máxima rapidez, eficiencia y seguridad. Realice sus operaciones desde la comodidad de su empresa o desde cualquier parte del mundo, las 24 horas del día, los 365 días del año.

El sistema ofrece un conjunto de servicios desarrollados con tecnología de última generación que revolucionarán su manejo administrativo y de tesorería.

Pago a Proveedores

Máxima Seguridad

Credinet Web cuenta con elementos electrónicos de seguridad de última generación, para que usted efectúe sus operaciones con la mayor confiabilidad:

  • Clave y nivel de acceso individual por usuario
  • Dispositivo Token que utiliza una clave dinámica para procesar las transacciones de manera más segura
  • Comunicación entre su Empresa y el Banco a través de una conexión segura y con algoritmo de encriptación SSL de 128 bits

Máyor rapidez

Credinet Web establece altas velocidades de conexión On Line, transmitiendo aproximadamente tres hojas de datos por segundo. Asimismo, el banco de Crédito BCP cuenta con una infraestructura de comunicaciones y equipos que utilizan el protocolo de Internet TCP/IP, el más eficiente y de mayor crecimiento en el mundo

Flexibilidad

Credinet Web ha sido diseñado de forma modular, para ofrecer un conjunto de servicios que pueden ser utilizados de acuerdo a sus necesidades. Asimismo, ha sido incorporado un nuevo concepto, que le permite a usted ser el propio Administrador del Sistema.

Facilidad de Uso

Credinet Web es un sistema amigable, fácil de entender y operar, porque utiliza la estructura de un ambiente Windows. Estamos seguros de que usted de familiarizará con el sistema rápidamente.

Asesoría Permanente
